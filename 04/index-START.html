Filter 함수

콜백함수의 조건에 해당하는 모든 요소로 새로운 배열을 생성하는 기능

콜백함수에는 3개의 매개변수(element, index, array)를 사용할 수 있다. 

const fifteen = inventors.filter(function(inventor){
        return ((inventor.year >= 1500) && (inventor.year < 1600));
      });
  
  //true나 truty한 값을 리턴하면 새로운 배열에 포함된다. (아무것도 리턴하지 않으면
  //false로 처리되므로 굳이 false를 명시적으로 리턴해줄 필요는 없음.)

Sort 함수

문자 정렬 : 콜백함수를 지정하지 않아도 아스키 문자 순서대로 자동 정렬됨. 

var fruits = ['orange', 'apple', 'banana'];
  
  fruits.sort(); //apple, banana, orange


숫자 정렬, 객체 정렬 : true/false (혹은 truthy/falsy) 리턴에 따라 정렬됨. 

var scores = [4, 11, 2, 10, 3, 1];
  scores.sort(); //1, 10, 11, 2, 3, 4 
                              //ASCII 문자 순서로 정렬됨. (숫자 크기대로 정렬 X)
  scores.sort((a, b) => (a - b)); //오름차순
  
  const ordered = inventors.sort((a, b) => a.year > b.year ? 1 : -1);

*정렬할 array의 element 개수가 2개 미만(1개, 0개)일 경우 sort is not a function 에러 발생

sort + split 메서드

문자열 전체가 아닌 일부를 기준으로 정렬하고 싶을 때

const alpha = people.sort((a, b) => {
        const [aLast, aFirst] = a.split(', ');
        const [bLast, bFirst] = b.split(', ');
        return (aLast > bLast) ? 1 : -1;
      });



Reduce 함수

누적값을 파라미터로 사용할 수 있다.

배열.reduce((누적값, 현잿값, 인덱스, 요소) => { return 결과 }, 초깃값);

예제1)

const totalYears = inventors.reduce((acc, cur) => {
        return acc += (cur.passed - cur.year);
      }, 0);


예제2)

const transportation = data.reduce((obj, item) => {
        if (!obj[item]) {
          obj[item] = 0;
        }
        obj[item]++;
        return obj;
      }, {})
